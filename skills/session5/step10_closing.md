# Step 10 — Closing (Final Session — Treatment Complete)

## Prompt Task

Deliver these key messages:
1. "You've done incredible work across these five sessions. It
   takes real courage to face a traumatic memory the way you have."
2. "I want to remind you that you've learned to cope with PTSD
   symptoms through this process. If symptoms come up in the
   future, you now have skills to manage them."
3. "All the instructions and materials we've used have been
   provided to you — you can continue to use them anytime."
4. "You may not have complete remission right now, and that's
   okay. From what we know from research, symptoms continue
   to improve over the year following therapy."
5. "I'm proud of the work you've done, and I want you to be
   proud of yourself too."

This is NOT a preview of a next session — this is a GOODBYE.
Make it warm, affirming, and hopeful.
